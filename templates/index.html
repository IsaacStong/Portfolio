What I do

Full-Stack

My venture into Full-Stack Engineering started with the goal of obtaining a certificate in Django development from Udemy. This morphed into a passion that has fueled a good number of my projects since. I love Full-Stack Engineering, getting to use the knowledge I've gained with my CS degree for back-end. Mixed with design and animating using a variety of things from plain old CSS to ReactJS for front-end has been an amazing experience. My current tech stack for Full-Stack includes HTML, CSS, JavaScript, ReactJS, JQuery, Python, Django, and NoSQL.

Game Design

My love for designing and creating video games started a child. I remember the first time I decided to create a video game was at eight years old. I quickly hit walls, but over the years have been able to overcome those obstacles. I have since been able to create a zombie themed pixel shooter, virtual escape room, and even had the joy of turning that virtual escape room into an immersive VR game. My hope is I one day get the chance to use these skills to create a truly breathtaking experience for others. My tech stack for Game Design includes Unity, Unreal Engine, Blender, C#, C++, and JavaScript for my web-based video games.

Machine Learning

Machine Learning has been a passion of mine ever since the start of college. From the very first CS course I took I remember thinking, "I want to create AI that will help change the world". This task has included learning about statistics, neural networks, and other machine learning principles. I have had the pleasure of using machine learning in some of my projects. With three courses under my belt I feel confident in my machine learning skills and often look to use these skills to automate and streamline my projects. My Machine Learning tech stack includes Anaconda, Python, Numpy, and Data Management.
